# Food Ordering Dashboard 🍔

A React-based Food Ordering Dashboard application built using React Hooks like useState, useEffect, useContext, useMemo, and useCallback.

## Features

- Food menu display
- Add items to cart
- Remove items from cart
- Dynamic total price calculation
- Responsive user interface

## Technologies Used

- React.js
- JavaScript
- CSS
- Vite

## React Hooks Used

- useState
- useEffect
- useContext
- useMemo
- useCallback

## How to Run

```bash
npm install
npm run dev
